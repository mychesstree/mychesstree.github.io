# ♟️ MyChessTree
MyChessTree is a chess repertoire trainer that combines the power of interactive tree mapping with a Spaced Repetition System (SRS). Designed for serious players, it helps you build, visualize, and memorize your opening lines with clinical precision.

![Dashboard Preview](https://raw.githubusercontent.com/mychesstree/mychesstree.github.io/main/public/logo.svg)


## 🛠️ Technology Stack

- **Frontend**: React 19, Vite, TypeScript
- **State & Auth**: Supabase
- **Chess Logic**: Chess.js, React-Chessboard, Stockfish WASM
- **Visualization**: D3.js, React-Calendar-Heatmap
- **Icons**: Lucide React
